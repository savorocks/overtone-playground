# Samples

All samples in this directory (except clave.wav) are from http://freesound.org and have
been placed in the public domain via the Creative Commons 0 License:

http://creativecommons.org/publicdomain/zero/1.0/

Many of the samples have been slightly modified and trimmed for use with
Sonic Pi.

Additionally, the samples have been further converted from .flac file format to .wav file format
because Overtone doesn't support loading of .flac files.
One of further aims of Overtone Playground will be to enable support
for various popular file formats.

Links to original samples:
